# Tech stack

Planned stack (free/hobby tiers)
- Next.js (App Router) — TypeScript (strict)
- Tailwind CSS for styling
- Supabase (Free) — Auth + Postgres + RLS
- TMDb API for movie metadata
- Vercel Hobby for hosting + Vercel Cron for scheduled jobs
- Vitest for unit tests
- Playwright for end-to-end tests
- GitHub Actions for CI

Rationale
- Fast developer experience, minimal infra management, and free tiers align with a personal app prototype.
